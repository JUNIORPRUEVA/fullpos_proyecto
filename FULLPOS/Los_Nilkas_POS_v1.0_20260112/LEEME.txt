INSTALACIÓN
-----------
1. Extraer todos los archivos en una carpeta de su preferencia
2. Hacer doble clic en "nilkas.exe" para iniciar la aplicación
3. NO eliminar ni mover ningún archivo de la carpeta

REQUISITOS DEL SISTEMA
-----------------------
- Windows 10 o superior (64 bits)
- 4 GB de RAM mínimo (8 GB recomendado)
- 500 MB de espacio libre en disco
- Resolución de pantalla: 1366x768 o superior

PRIMER USO
----------
Usuario por defecto: admin
Contraseña por defecto: admin

IMPORTANTE: Cambiar la contraseña después del primer inicio de sesión

CARACTERÍSTICAS PRINCIPALES
----------------------------
✓ Gestión completa de ventas y cotizaciones
✓ Control de inventario y productos
✓ Administración de clientes
✓ Sistema de créditos y préstamos
✓ Reportes y estadísticas detalladas
✓ Impresión de tickets térmicos
✓ Configuración personalizable del negocio
✓ Gestión de múltiples usuarios y permisos
✓ Backup automático de datos
