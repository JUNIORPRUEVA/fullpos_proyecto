🚀 CÓMO USAR
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1️⃣ EN CONFIGURACIÓN
   └─ Settings > TEMA > Seleccionar uno de los 3 temas

2️⃣ CAMBIAR TEMA EN CÓDIGO
   ```dart
   final notifier = ref.read(appThemeProvider.notifier);
   await notifier.setTheme(AppThemeEnum.proPos);
   ```

3️⃣ OBTENER TEMA ACTUAL
   ```dart
   final currentTheme = ref.watch(appThemeProvider);
   // Retorna: AppThemeEnum (original, azulBlancoNegro, proPos)
   ```

4️⃣ USAR COLORES DEL TEMA
   ```dart
   Text(
     'Contenido',
     style: Theme.of(context).textTheme.bodyLarge,
   )
   ```
