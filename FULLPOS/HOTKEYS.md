# 🎯 Atajos de Teclado - LOS NILKAS POS

## 🌐 Atajos Globales (Funcionan en toda la aplicación)

| Atajo | Acción | Descripción |
|-------|--------|-------------|
| `Ctrl + Shift + F` | **Toggle Fullscreen** | Activa/desactiva el modo pantalla completa POS |
| `ESC` | **Cerrar Diálogos** | Cierra diálogos abiertos o navega hacia atrás |
| `Ctrl + Q` | **Cerrar Aplicación** | Cierra la aplicación (con confirmación) |

---

## 🛒 Atajos de Ventas (Solo en Pantalla de Ventas)

### 🔍 Búsqueda y Selección

| Atajo | Acción | Descripción |
|-------|--------|-------------|
| `F2` | **Enfocar Búsqueda** | Coloca el cursor en el buscador de productos |
| `F3` | **Seleccionar Cliente** | Abre el selector de clientes |
| `F4` | **Nuevo Cliente** | Abre el formulario para crear un cliente nuevo |
| `Enter` | **Agregar Producto** | Agrega el producto seleccionado al carrito |

### 📝 Gestión de Items

| Atajo | Acción | Descripción |
|-------|--------|-------------|
| `Clic Simple` | **Seleccionar Item** | Selecciona un item del carrito |
| `Doble Clic` | **Editar Item** | Abre el diálogo de edición del item |
| `F7` | **Aplicar Descuento** | Abre diálogo de descuento para el item seleccionado |
| `Ctrl + Backspace` | **Eliminar Item** | Elimina el item seleccionado del carrito |
| `+` o `Shift + =` | **Aumentar Cantidad** | Incrementa la cantidad del item seleccionado |
| `-` | **Disminuir Cantidad** | Reduce la cantidad del item seleccionado |

### 💰 Pago y Finalización

| Atajo | Acción | Descripción |
|-------|--------|-------------|
| `F9` | **Abrir Pago** | Procesa el pago/cobro de la venta actual |
| `F12` | **Finalizar Venta** | Finaliza y confirma la venta actual |

## ⚙️ Configuración del Modo POS

### Activar Modo Pantalla Completa

1. Ve a **Configuración** (Settings)
2. Busca la opción **"Modo POS Pantalla completa"**
3. Activa el switch
4. La aplicación entrará automáticamente en modo fullscreen

### Características del Modo Fullscreen:

- ✅ Ventana maximizada sin bordes
- ✅ Barra de tareas oculta cuando la app está enfocada
- ✅ Siempre visible en taskbar para facilitar Alt+Tab
- ✅ No se puede redimensionar en modo fullscreen
- ✅ Se guarda la preferencia entre sesiones

### Salida de Emergencia

Si necesitas salir del modo fullscreen:

1. **Método Principal**: Presiona `Ctrl + Shift + F`
2. **Método Alternativo**: Ve a Configuración y desactiva "Modo POS Pantalla completa"
3. **Cerrar App**: Presiona `Ctrl + Q` y confirma

## 💡 Tips y Mejores Prácticas

### Workflow Óptimo para Ventas:

1. `F2` → Buscar producto
2. `Enter` → Agregar al carrito
3. Repetir 1-2 para más productos
4. `Clic` en item → Seleccionar para modificar
5. `+/-` → Ajustar cantidad rápidamente
6. `F7` → Aplicar descuentos si es necesario
7. `F3` → Asignar cliente (opcional)
8. `F9` o `F12` → Procesar pago

### Navegación Rápida:

- Usa `F2` constantemente para volver al buscador
- Usa `F3` para cambiar de cliente sin el mouse
- Usa `Ctrl + Backspace` para eliminar productos incorrectos
- Doble clic en items para editar cantidad y descuento

### Modo Kiosk:

- Activa fullscreen para una experiencia inmersiva
- La barra de tareas se oculta automáticamente
- Usa `Ctrl + Shift + F` para personal autorizado
- `Ctrl + Q` solo para administradores

## 🔒 Seguridad

- **Ctrl + Q** requiere confirmación antes de cerrar
- **Ctrl + Shift + F** siempre disponible para salir de fullscreen
- No hay "lock mode" permanente - siempre hay salida
- Fullscreen NO bloquea el sistema operativo

## 🐛 Solución de Problemas

### Los atajos no funcionan:

1. Asegúrate de estar en la pantalla correcta
2. Verifica que no haya un diálogo abierto
3. Cierra y reabre la aplicación
4. Verifica que tu teclado no tenga `Fn Lock` activado

### Fullscreen no se activa:

1. Verifica permisos de la aplicación
2. Cierra otras aplicaciones en fullscreen
3. Reinicia la aplicación
4. Usa `Ctrl + Shift + F` para toggle manual

### No puedo salir de fullscreen:

1. Presiona `Ctrl + Shift + F`
2. Si no funciona, presiona `Alt + F4`
3. Usa `Ctrl + Alt + Del` para abrir Task Manager
4. Como último recurso, reinicia el equipo
