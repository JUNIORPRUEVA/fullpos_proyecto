💻 CÓDIGO - ANTES vs DESPUÉS
═════════════════════════════════════════════════════════════════════════════

❌ ANTES (Viejo):
──────────────────
   Text(
     'Hola',
     style: TextStyle(color: Color(0xFF00796B)),  // Hardcodeado
   )

✅ DESPUÉS (Nuevo):
──────────────────
   Text(
     'Hola',
     style: Theme.of(context).textTheme.bodyMedium,  // Del tema
   )
   
   // El color viene automáticamente del tema seleccionado
   // Si cambias a otro tema, el color se actualiza sin cambiar nada

🎮 CÓMO USAR
═════════════════════════════════════════════════════════════════════════════

EN CONFIGURACIÓN:
   Abre: Configuración > TEMA (ícono de paleta)
   Selecciona: Uno de los 3 temas
   Resultado: La app cambia al tema seleccionado

EN CÓDIGO (para cambiar tema):
   ────────────────────────────
   final notifier = ref.read(appThemeProvider.notifier);
   await notifier.setTheme(AppThemeEnum.proPos);  // Cambiar a POS
   
   // Atajos disponibles:
   await notifier.setOriginal();           // Volver a original
   await notifier.setAzulBlancoNegro();    // Cambiar a azul
   await notifier.setProPos();             // Cambiar a POS

EN WIDGETS (para usar colores/estilos):
   ────────────────────────────────────
   Text(
     'Mi Texto',
     style: Theme.of(context).textTheme.bodyMedium,
   )
   
   Container(
     color: Theme.of(context).colorScheme.primary,
   )

🎓 CONCEPTO CLAVE: Theme.of(context)
═════════════════════════════════════════════════════════════════════════════

Theme.of(context) retorna el ThemeData actual.

Eso significa:
   - Si el usuario selecciona "Tema Original"
     → Theme.of(context) retorna el ThemeData del Tema Original
   
   - Si el usuario selecciona "Tema POS"
     → Theme.of(context) retorna el ThemeData del Tema POS
   
   - Automáticamente, sin que hagas nada

Ejemplo:
   Text(
     'Hola',
     style: Theme.of(context).textTheme.bodyMedium,
   )
   
   ↓ Si tema es Original ↓
   style = {color: #1F2937, fontSize: 14, ...}  // Colores Original
   
   ↓ Si tema es Azul ↓
   style = {color: #111827, fontSize: 14, ...}  // Colores Azul
   
   ↓ Si tema es POS ↓
   style = {color: #111827, fontSize: 14, ...}  // Colores POS
