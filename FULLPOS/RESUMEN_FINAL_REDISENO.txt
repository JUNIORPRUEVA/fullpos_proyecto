## 🎯 CAMBIOS IMPLEMENTADOS

### 3️⃣ NUEVOS CAMPOS EN TicketLayoutConfig

```dart
final String headerAlignment;    // 'left' | 'center' | 'right'
final String detailsAlignment;   // 'left' | 'center' | 'right'
final String totalsAlignment;    // 'left' | 'center' | 'right'
```

**Defaults profesionales:**
- headerAlignment: `'center'`
- detailsAlignment: `'left'`
- totalsAlignment: `'right'`

---

### 3️⃣ NUEVAS FUNCIONES EN TicketBuilder

#### `alignText(text, width, align)`
Alinea texto genéricamente respetando maxCharsPerLine.

#### `sepLine(width, [char = '-'])`
Crea líneas separadoras del ancho especificado.

#### `totalsLine(label, value, width, align)`
Crea línea de total con alineación configurable.

## 🎨 VISUAL: ANTES vs DESPUÉS

### ❌ ANTES (Líneas largas, decoraciones excesivas)
```
================================================
         FULLTECH, SRL
      RNC: 133080206 | Tel: 829-555-1234
           Centro Balber 9
================================================

FACTURA                     FECHA: 29/12/2025
                           TICKET: #000001
```

### ✅ DESPUÉS (Limpio, elegante, configurable)
```
========================================
         FULLTECH, SRL
RNC: 133080206 | Tel: +1(829)531-8442
        Centro Balber 9
========================================

FACTURA                 FECHA: 29/12/2025
                        TICKET: #DEMO-001
```

**✨ Más limpio, más elegante, mejor alineado**

## 💻 EJEMPLO DE USO

### Configuración por Defecto (Listo para usar)
```dart
final config = TicketLayoutConfig.professional80mm();
final builder = TicketBuilder(layout: config, company: company);
final text = builder.buildPlainText(data);
```

### Personalizar Alineación
```dart
final config = TicketLayoutConfig(
  headerAlignment: 'center',      // Encabezado centrado
  detailsAlignment: 'left',       // Detalles a izquierda
  totalsAlignment: 'right',       // Totales a derecha
);
```

### Las 3 Nuevas Funciones
```dart
builder.alignText('TÍTULO', 42, 'center');
builder.sepLine(42, '-');
builder.totalsLine('TOTAL', 'RD$ 1000', 42, 'right');
```
