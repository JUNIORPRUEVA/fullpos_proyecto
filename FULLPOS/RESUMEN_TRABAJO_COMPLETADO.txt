💡 LECCIÓN TÉCNICA IMPORTANTE

El patrón INCORRECTO que causó el problema:
```
Navigator.pop(context);  // Cierra pantalla
await _loadQuotes();      // Intenta actualizar pantalla cerrada
```

Resultó en contexto inválido → setState() sin widget válido → Pantalla negra

El patrón CORRECTO aplicado:
```
await _loadQuotes();      // Actualiza en contexto VÁLIDO
// No hay Navigator.pop() que interfiera
```

Resultado: Pantalla actualizada correctamente ✨

Esta lección aplica a cualquier situación en Flutter donde necesites
actualizar datos en la pantalla actual sin cerrarla.
