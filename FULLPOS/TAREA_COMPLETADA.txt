🎓 LECCIÓN TÉCNICA IMPORTANTE

El patrón que CAUSÓ el problema:
────────────────────────────────
  try {
    await operacion();
    Navigator.pop(context);  // ← Cierra la pantalla
    await _loadQuotes();     // ← Intenta actualizar pantalla cerrada
    // Resultado: CONTEXTO INVÁLIDO → PANTALLA NEGRA
  }

El patrón CORRECTO que lo arregló:
──────────────────────────────────
  try {
    await operacion();
    await _loadQuotes();     // ← Actualiza en contexto VÁLIDO
    // NO hay Navigator.pop()
    // Resultado: PANTALLA ACTUALIZADA CORRECTAMENTE
  }

Esta lección aplica a CUALQUIER situación en Flutter donde necesites
actualizar datos sin cerrar la pantalla.
